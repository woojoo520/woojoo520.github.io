神经网络的典型训练程序如下：

  • 定义一些具有可学习参数（或权重）的神经网络

  • 迭代输入数据集

  • 通过网络处理输入

  • 计算损失（输出距离正确多远）

  • 将渐变传播回网络参数

  • 通常使用简单的更新规则更新网络权重
