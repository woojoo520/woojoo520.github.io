128. 最长连续序列

要求算法的时间复杂度为 O(n)。

思路：

哈希表 + 线性空间的构造

我们先将所有元素放进一个Hash set中，然后遍历num_set中的每一个元素，如果它前面一个数字没有在num_set中，那么它一定不是一个连续数组的开始，则跳过。如果是连续数组的开始，那么就需要一种往后找连续的元素是否存在，然后记录下最长的数列长度。
